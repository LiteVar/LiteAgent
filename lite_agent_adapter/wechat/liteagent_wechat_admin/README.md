## ðŸ—ºï¸ Routing Architecture

### Main Application Routes (Authentication Required)
These routes are wrapped by `AppLayout` and require the user to be authenticated through WeChat Official Account:

| Route Path | Component | Description |
|------------|-----------|----------------------------------------------------------------|
| `/` | Home | Dashboard - Displays system overview and key metrics |
| `/wechat` | WechatManagement | Bot Management - Manage bindings between WeChat bots and LiteAgent Agents |
| `/system` | SystemInfo | System Information - System status monitoring and configuration |

### Error Handling
| Route Path | Component | Description |
|------------|-----------|---------------------------------------|
| `*` | NotFound | 404 Page - Handles unmatched route requests |

### Authentication Protection Mechanism
- **Global Authentication Check**: The `AppLayout` component checks for a token in `localStorage`.
- **Automatic Redirection**: Unauthenticated users are automatically redirected to `/signin`.
- **Token Management**: The API client's authentication header is automatically set after a successful login.
- **State Listening**: Listens for changes in `localStorage` to support synchronization across multiple tabs.

## ðŸ“¦ Installation and Quick Start

### Prerequisites

Before you begin, ensure your development environment meets the following requirements:

- **Node.js**: 18.x or higher (Node.js 20.x+ recommended)
- **Package Manager**: npm, yarn, or pnpm
- **WeChat Official Account Platform Account**: For creating enterprise applications and obtaining OAuth2 configuration.

### Quick Start

#### 1. Clone the project

```bash
git clone git@gitlab.litevar.com:litevar/lite-agent/lite-agent-wechat-admin.git
cd lite-agent-wechat-admin
```

#### 2. Install dependencies

```bash
# Using npm
npm install

# Using yarn
yarn install

# Using pnpm
pnpm install
```

> ðŸ’¡ If you encounter dependency installation issues, try using the `--legacy-peer-deps` flag.

#### 3. Start the development server

```bash
# Using npm
npm run dev

# Using yarn  
yarn dev

# Using pnpm
pnpm dev
```

The project will start at `http://localhost:5173`. On the first visit, you will be automatically redirected to the WeChat Official Account login page.

### API Usage
The project uses a type-safe client auto-generated by OpenAPI:

```typescript
import { client } from '../api';

// Authentication header is automatically included
const response = await client.getRobots();
```

### Component Development Guidelines
- Write all components in TypeScript.
- Follow the React Hooks pattern.
- Use Tailwind CSS for styling.
- Adhere to the single-responsibility principle for components.
